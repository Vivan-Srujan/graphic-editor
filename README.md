# Graphic Editor

A simple and interactive Graphic Editor project developed using programming concepts and graphical tools. This project allows users to create, edit, and manage graphical elements efficiently.

## Features

* Draw basic shapes
* Edit graphical objects
* User-friendly interface
* Save and manage projects
* Simple and lightweight design

## Technologies Used

* Python / C / C++ *(update according to your project)*
* VS Code
* Git & GitHub

## Project Structure

```bash
graphic-editor/
│
├── src/
├── assets/
├── README.md
└── main.py
```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Vivan-Srujan/graphic-editor.git
```

2. Open the project folder:

```bash
cd graphic-editor
```

3. Run the project:

```bash
python main.py
```

## Usage

* Open the application
* Select drawing tools
* Create or edit graphics
* Save your work
